# 🚀 Professional AI/ML Portfolio | Sai Krishna Gummadidala

Welcome to my professional portfolio! This project is a high-performance, visually stunning, and fully responsive web application designed to showcase my journey as an AI & ML Engineering student.

---

## 🛠️ Tech Stack (The "DStack")

I've used a modern and powerful stack to ensure speed, scalability, and a premium user experience:

*   ⚛️ **React 19** – The core library for building a dynamic and component-based UI.
*   ⚡ **Vite** – Next-generation frontend tooling for an ultra-fast development experience.
*   🎨 **Tailwind CSS** – Utility-first CSS framework for rapid and precise styling.
*   ✨ **Framer Motion** – Production-ready motion library for smooth, fluid animations.
*   🛡️ **TypeScript** – Adding static typing to ensure code reliability and maintainability.
*   🧩 **Lucide React** – A beautiful and consistent icon set for enhanced visual communication.

---

## ✨ Key Features

*   🌌 **Deep Navy Premium Theme** – A custom-crafted dark aesthetic that feels professional and modern.
*   📱 **Fully Responsive** – Seamless experience across mobile, tablet, and desktop devices.
*   🎭 **Smooth Transitions** – Elegant entrance animations and scroll-triggered effects.
*   📂 **Project Showcase** – Detailed cards highlighting my technical work and the tools used.
*   🛠️ **Technical Toolbox** – A categorized display of my expertise in languages, web tech, and AI/ML.
*   🎓 **Academic Timeline** – A clean visualization of my educational background.
*   ✉️ **Interactive Contact** – A user-friendly form and direct links to connect with me.
